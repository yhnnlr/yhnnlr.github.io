Dropper
Game Jam
Mini-Game / C# / Unity
2 days - 2024
Dropper is a Mini-Game inspired by the Minecraft map 'The Dropper'. A suicidal person jumps from a building and hallucinates as he falls.
The Game Jam's theme was 'Deeper and deeper', with the optional constraint 'Black and white'.
This game was created for CreaJam 2024, which began on March 15, 2024 at 6pm and ended on March 17, 2024 at 3pm.
I made this game alone, so I made everything in the game except for the assets I found for free in the Unity Asset Store.
As it was my first Game Jam, I managed my time pretty poorly. I spent very little time on the gameplay itself, with the level-design and the shaders taking up most of my time. I'm still happy with the result, but in my opinion it could've been way better.
Software used : Visual Studio, Unity, Trello, Github Desktop
Skills involved : Gameplay Programming, Game Design, Level Design, Cinematics, Object-Oriented Programming, Shaders
