CreaRaider
Solo Project
Platformer/Puzzle / C# / Unity

This was my first 3D project, made in Unity. Learning to use a game engine as well as learning a new language was pretty tough, but I believe I did a great job still. This was a milestone in my game developing career!
Software used : Visual Studio, Unity, Tortoise SVN
Skills involved : Gameplay Programming, UI
