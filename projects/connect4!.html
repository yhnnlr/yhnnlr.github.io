Connect 4!
Solo Project
Board Game / Lua / LÖVE
4 days - January 2022
Connect 4! is a simple game of Connect 4 with local multiplayer (same mouse). The visual effects react on the music!
This was the first project I was proud of. I spent a lot of time researching the LÖVE framework online so that the game would look exactly as I'd imagined it. This project is the one that made me love developing video games, so even if it's not much, it holds a special place in my heart.
Software used : ZeroBrane Studio, LÖVE
Skills involved : Gameplay Programming, Sound, Grids, UI
