Doodle Jump!
Solo Project
Arcade Game / C / CSFML
1 week - Febuary 2022
Doodle Jump! is, as the name suggests, a replica of the game Doodle Jump!.
This was our first project using CSFML. I tried to make the game as close as possible to the original, and I'm pretty happy with the result.
Software used : Visual Studio, CSFML, Tortoise SVN
Skills involved : Gameplay Programming, Procedural Generation, UI
