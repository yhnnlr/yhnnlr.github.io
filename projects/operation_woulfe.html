Operation Woulfe
Solo Project
Rail-Shooter / Lua / LÖVE
1 week - October 2021
Operation Woulfe is a replica of the game Operation Wolf. Shoot every enemy on screen to win!
This was my first ever programming project. Even at the time I wasn't happy with the result, I believe I could've done way better as the game softlocks itself when you miss an enemy and there is no win screen.
Software used : ZeroBrane Studio, LÖVE, Tortoise SVN
Skills involved : Gameplay Programming, UI
