CreaMan
Solo Project
Arcade Game / C++ / SFML
1 week - November 2022
CreaMan is a custom version of the famous game Pac-Man.
This was our very first project using SFML. We were asked to do a simple replica of Pac-Man, but I found the assignment a little boring so the edgy boy that I was decided to add a little twist to it. In the end, I had a lot of fun doing it!
Software used : Visual Studio, SFML, Tortoise SVN, Paint.net
Skills involved : Gameplay Programming, AI, Camera, Sound
