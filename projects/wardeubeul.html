Wardeubeul
Solo Project
MMORPG / C++ / MySQL
2 weeks - January 2024
Wardeubeul is a MMORPG in command prompt. Earn exp by doing quests, send messages to other players, join a guild and become the best player on the server!

This game is a bit tricky to install, be sure to read ReadMe.txt in the project's files.
Language: French only!
I had a lot of fun making this project. This was my first time using MySQL (and using a database in general).
This made me want to create a bigger version with a database hosted online. Who knows, maybe in the future?
Software used : Visual Studio, WampServer, PhPMyAdmin, MySQL, Tortoise SVN
Skills involved : Database
